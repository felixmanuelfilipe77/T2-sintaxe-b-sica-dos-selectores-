Sintaxe Básica dos Seletores CSS

Aprenda a selecionar elementos HTML e aplicar estilos com CSS.

1. Seletor de Elemento

Seleciona todos os elementos de um tipo específico.

Exemplo: <p>Este parágrafo usa o seletor de elemento <p></p>

2. Seletor de Classe

Seleciona todos os elementos com uma determinada classe.

Exemplo: .classe-exemplo { color: blue; }

Este parágrafo é estilizado pela classe "classe-exemplo".

3. Seletor de ID

Seleciona um elemento único com um ID específico.

Exemplo: #id-exemplo { font-weight: bold; color: red; }

4. Seletor de Grupo

Seleciona múltiplos elementos de uma vez.
